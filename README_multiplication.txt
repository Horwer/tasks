Алгоритм рабботы:
1. Пользователь вводит в текстовые поля множители и нажимает на кнопку "Перемножить".
2. Введенные множители проверяются на соответствие формату.
3. Множители сохраняются в массив.
4. Массив передается в качестве аргумента в функцию умножения.
5. Первый элемент массива сохраняется в переменную в которой хранится результат.
6. Каждый последующий элемен умножается на переменную результата и сохраняется в ней.
7. Результат перемножения выводится пользователю.
